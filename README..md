Proyecto: Registro de usuario - U4
## Descripción
Este proyecto implementa un formulario de registro de usuario con validación del lado del cliente usando JavaScript.
El formulario verifica que los datos ingresados cumplan con los requisitos antes de permitir el envío.

Se combinan validaciones HTML nativas y validaciones manuales con JavaScript.

## Funcionalidades
- Validación de nombre completo
- Validación de correo electrónico
- Validación de contraseña (mínimo 8 caracteres, una mayúscula y un número)
- Confirmación de contraseña
- Validación opcional de teléfono
- Mensajes de error personalizados
- Validación en tiempo real
- Mensaje de registro exitoso
- Indicador de fortaleza de contraseña

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript (ES6)
- DOM
- Constraint Validation API

## Cómo ejecutar el proyecto
1. Clonar el repositorio o descargar los archivos.
2. Abrir la carpeta en tu editor de codigo .
3. Ejecutar el archivo `index.html` usando Live Server o abrirlo en el navegador.
